Informazioni sul componente

La CPU (Central Processing Unit) e' l'unita' di elaborazione centrale del sistema. Controlla tutte le funzioni della macchina ed esegue tutte le funzioni di calcolo. E' composta da:

  • ALU (Unita' con funzioni Aritmetiche Logiche)
  • Control Unit (Unita' di Controllo)
  • Registri (piccole aree di memoria)
  • Cache interna (memoria utile a una rapida esecuzione di istruzioni)
  • Logica di controllo (trasforma sia i comandi, sia i segnali dall'esterno in segnali elettrici)
  • Memory Interface (comunicazione e trasmissione dei dati tra unita' esterne e processore stesso)
